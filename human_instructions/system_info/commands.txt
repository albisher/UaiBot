## Examples

### English Examples
1. Check system status:
   ```
   show system status
   ```

2. Monitor CPU usage:
   ```
   show CPU usage
   ```

3. Check memory:
   ```
   show memory usage
   ```

4. View disk space:
   ```
   show disk usage
   ```

### Arabic Examples
1. التحقق من حالة النظام:
   ```
   عرض حالة النظام
   ```

2. مراقبة استخدام المعالج:
   ```
   عرض استخدام المعالج
   ```

3. التحقق من الذاكرة:
   ```
   عرض استخدام الذاكرة
   ```

4. عرض مساحة القرص:
   ```
   عرض استخدام القرص
   ```

## Notes
- All system information should be updated in real-time
- Metrics should be displayed in appropriate units (bytes, percentages, etc.)
- Historical data should be available for trend analysis
- Critical thresholds should be highlighted
- Both English and Arabic numbers should be supported
- Time stamps should be in the local timezone
- Error states should be clearly indicated
- Performance impact should be minimized
- Data should be properly formatted for readability
- Security-sensitive information should be properly protected
